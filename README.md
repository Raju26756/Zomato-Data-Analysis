# Zomato Data Analysis Project

## 📊 Tools Used
- Excel
- SQL
- Power BI
- Tableau

## 📌 Project Overview
This project analyzes Zomato restaurant data to identify key trends in ratings, pricing, cuisines, and online delivery patterns.

## 📈 Key Analysis Performed
- Built data model and calendar table for time-based analysis  
- Converted cost values into USD for consistency  
- Analyzed restaurant distribution by city and country  
- Evaluated ratings, pricing buckets, and delivery trends  
- Created KPIs for business insights  

## 📊 Dashboard Insights
- City-wise restaurant distribution  
- Online delivery & table booking percentage  
- Cuisine-wise analysis  
- Rating and pricing trends  

## 📂 Files Included
- Excel Dashboard  
- SQL Queries  
- Power BI Dashboard  
- Tableau Dashboard  
- Project Presentation  

## 🚀 Conclusion
This project provides insights into customer preferences and restaurant performance, helping in data-driven decision-making.
